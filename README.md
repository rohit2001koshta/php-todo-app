# PHP To-Do List App

A simple and beginner-friendly To-Do List application built using **PHP + MySQL**.

## Features
- Add tasks
- Mark tasks as completed
- Delete tasks
- View all tasks
- Bootstrap UI
- Simple CRUD operations

## Requirements
- PHP 7+ / PHP 8+
- MySQL
- XAMPP or WAMP
- Web browser

## How to Install
1. Download or clone the repository.
2. Move the folder to `htdocs` (XAMPP).
3. Import `database.sql` into phpMyAdmin.
4. Start Apache & MySQL.
5. Open `http://localhost/php-todo-app/` in your browser.

## Project Structure
- `index.php` – Display task list
- `add.php` – Add new tasks
- `update.php` – Mark tasks as complete
- `delete.php` – Delete tasks
- `config.php` – Database connection file
- `database.sql` – MySQL database file
